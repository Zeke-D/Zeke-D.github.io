Reddiment Analysis

This was a project for Bostonography, a class dedicated to analyzing Boston through computer science data analytics. We actively used R, GIS, and Python to interpret public access datasets provided by the Boston City government. For this project, I decided to determine if there was a correlation between the time of year and the general attitude of Reddit users over different subreddits. My work is on pages 6-12.

Kids For Code

This was a personal project I had really wanted to get going during my free time in high school. I realized that I loved programming in part because I had been teaching myself it since 5th grade. However, I noticed there weren't a lot of opportunities for kids in middle school to learn coding. So I decided to start an outreach program to our local 4th-6th grade middle school to see if I could come teach lessons during my lunch period some days a month. It worked, and was a lot of fun.
